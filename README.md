<h1>Cronômetro</h1>
<p>Está é uma aplicação web de um cronômetro, com as funções "Começar" (Inicia a contagem), "Pausar" (Pausa ou recomeça a contagem) e "Parar" (Reseta e paralisa a contagem)</p>
